# PosTech.TechChallenge.Phaseive

## Introdução

Esse projeto tem como objetivo demonstrar a utilização de todos os conhecimentos adquiridos durante a **Fase 5** na Pós Graduação da FIAP em **Desenvolvimento .NET e Azure**.

## Requisitos

O Tech Challenge desta fase será a demonstração da utilização do **Kubernetes** e conceito de **LGPD**. Os requisitos são:

- Publicação em um Cluster local do Kubernetes.
- Criptografia de dados sensíveis e pessoais no Mongo DB.

## Resolução

**Cenário**: Uma API simples com cadastro de clientes com dados pessoais sensíveis.

O objetivo era explorar um cenário real, onde além da publicação da aplicação no Cluster Kubernetes, também fosse configurado a infraestrutura do Mongo DB.
Como qualquer armazenamento de dados sensíveis, também gostariamos de explorar situações de adoção de técnicas para atender a LGPD.
